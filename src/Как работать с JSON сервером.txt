1)создаёшь простой файлик - например db.json .
2)устанавливаешь json сервер - npm i -g json-server
3)с помощью команды json-server - указываешь путь к базе данных 
  json-server db.json --port 4200 --watch
  --port - указать порт для запуска
  --watch - флаг для слежки изменений за базой данных (db.json).
